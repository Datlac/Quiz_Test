2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:


4. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000).

## üåê How to Deploy (Make it Public)

The easiest way to deploy this Next.js app is to use [Vercel](https://vercel.com/new).

1. **Push to GitHub**:
   Make sure this project is pushed to a repository on your GitHub account.

2. **Connect to Vercel**:
   - Go to [Vercel.com](https://vercel.com) and sign up/login (you can use your GitHub account).
   - Click **"Add New Project"**.
   - Select your `personalized-quiz-master` repository.
   - Click **"Deploy"**.

3. **Done!**:
   Vercel will automatically build your site and give you a live URL (e.g., `https://personalized-quiz-master.vercel.app`) that you can share with anyone.
